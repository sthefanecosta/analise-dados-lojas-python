# 📊 Desafio: Consultoria de Dados para o Varejo

Este projeto foi desenvolvido como um exercício prático em análise de dados, com o objetivo de apoiar um empresário em uma decisão estratégica crucial: **determinar qual unidade de negócio descontinuar (vender).**

O foco foi transformar um dataset de vendas em uma recomendação objetiva, fundamentada em dados reais de performance.

---

## 💡 A Jornada do Projeto

Utilizei **Python** para explorar o desempenho de quatro unidades, analisando métricas-chave que ajudaram a embasar a estratégia:

* **Faturamento:** Identificamos que a **Loja 4** possui o menor faturamento dentre todas as unidades, atingindo cerca de R$ 1.38 milhões.


* **Logística (Frete):** Apesar de possuir o menor custo de frete da rede, esse fator não se traduziu em vendas mais robustas para a Loja 4.


* **Satisfação:** A avaliação dos clientes foi bastante homogênea em todas as unidades (~4.0). Isso indica que os consumidores percebem a operação positivamente, mas a unidade enfrenta dificuldades específicas de escala e volume de vendas.


---

## 🛠️ Ferramentas Utilizadas

* **Python:** Linguagem principal para análise e processamento.
* **Pandas:** Essencial para limpeza, cálculos de médias, somatórios e agrupamentos.
* **Matplotlib:** Biblioteca utilizada para criar as visualizações que reforçam as conclusões.

---

## 🚀 Conclusão e Recomendação

A recomendação técnica é a **venda da Loja 4**.

Apesar das vantagens do menor custo logístico, essa unidade não consegue gerar vendas suficientes para justificar sua continuidade frente às demais lojas que apresentam resultados mais sólidos. A estratégia sugerida é otimizar os recursos, direcionando investimentos para as unidades com maior potencial de escala, como a **Loja 1**, garantindo maior retorno financeiro e eficiência operacional.
